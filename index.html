Gaming Console

Insert cartridge · Play with your own numbers & words

Pick a pastel cartridge above—each game is built from real puzzle logic (queues, stacks, arrays, recursion) and needs your input to run.

  • Sudoku — fill cells; solver runs recursive backtracking with constraint checks.
  • Minesweeper — reveal cells; counts sum neighboring mines; zero-cells flood-fill open the region.
  • Crossword — type letters; answers checked against a keyed grid map.
  • Bracket Stack — type code-like brackets; a stack pushes/opens and pops/matches.
  • Sort Studio — paste numbers; insertion sort or merge sort (divide & conquer + merge).
  • Binary Search — sorted list + target you choose; each step narrows [lo, hi].

Sudoku · Backtracking

Enter digits 1–9. Empty cells stay blank. Solve tries candidates depth-first with row, column, and 3×3 box checks.

Minesweeper · Adjacency counts

Left-click to reveal a cell. Right-click to plant/remove a flag. The first tap never hits a mine — mines are placed after that, away from your opening square. Open cells show how many mines touch them (0–8); empty regions expand in one sweep.

Crossword · Grid map

Letters keyed by row,column in an answer map—same idea as a sparse dictionary over coordinates.

Bracket Stack · LIFO validation

Type ( ) [ ] { } and any other characters—only brackets matter. The algorithm scans left-to-right: opening brackets push onto a stack; closing brackets pop and must match.












Sort Studio · Array algorithms


Comma-separated integers (e.g. 5, 2, 9, 1). Insertion sort builds a sorted prefix in-place. Merge sort splits halves recursively and merges sorted runs.

























Binary Search · Divide interval


Must be sorted ascending. You provide the array and target; each step compares mid and halves the search range [lo, hi].
